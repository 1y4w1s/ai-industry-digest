## 第二部分：品牌重塑 — Signal

### 2.1 品牌命名

| 要素 | 内容 |
|------|------|
| **名称** | **Signal** |
| **含义** | 从信息噪音中筛出真实信号 |
| **Slogan** | "Signal from the noise" |
| **定位** | 每日 AI 行业信号简报 |
| **Logo 形式** | 纯文字 Logo，Source Serif 4 衬线字体，全大写或首字母大写 |
| **Logo 变体** | `S` 字母 + 一条细水平线（象征信号波形） |

### 2.2 设计理念

> 不靠颜色吸引眼球，靠内容本身。

参考气质：《纽约时报》/ Monocle / Stratechery

核心原则：
- **黑白即品牌** — 没有品牌色，排版+留白就是视觉语言
- **内容先行** — 不装饰、不花哨，每个元素都为阅读服务
- **克制即高级** — 减少圆角、减少阴影、减少一切「工具感」装饰

### 图标规范 — 全站统一规则

**核心理念：全站所有图标无色，只有黑白灰。** 唯一允许有颜色的地方是重要性指示点（红/金/灰圆点）。

**样式：轮廓 ↔ 实心切换**

| 状态 | 样式 | 色值 |
|------|------|------|
| **未选中/默认** | 空心轮廓 `stroke-width=1.5` | `#8C9096` |
| **hover** | 同空心轮廓 | `#686C72`（加深） |
| **选中/活跃** | 实心填充 `fill` | `#1A1C1E` |

**适用范围（全站）：**

| 位置 | 图标 | 说明 |
|------|------|------|
| 侧边栏导航 | 首页/搜索/收藏/历史/统计 | ⚠️ 首项「今日日报」默认选中→实心，其余空心 |
| Header 搜索 | 放大镜 | 固定在搜索框内，灰色 |
| 文章操作栏 | PDF/收藏/反馈 | 默认空心，点击后实心（如收藏） |
| 右侧面板 | 热点/趋势/Top5/来源 | 无图标，纯文字 |
| 筛选栏重要性 | 红/金/灰圆点 | **唯一允许彩色的地方** |
| AI 气泡 | 聊天气泡 | 灰色轮廓，hover 加深 |
| 关闭/箭头 | 各种交互按钮 | 统一灰色轮廓 |

**图标集选择：Lucide Icons**
- 理由：开源、一致性高、支持 `stroke` / `fill` 属性切换，可 tree-shaking
- 替代方案：自绘极简几何 SVG（Linear 风格），每条边 2px 圆头描边

**不使用的图标风格：**
- ❌ Heroicons（所有 AI 工具都在用，AI 味重）
- ❌ FontAwesome（太厚重）
- ❌ emoji 当图标（不严肃）
- ❌ 任何彩色图标（与黑白风格冲突）
- ❌ 渐变/双色图标

**实现提示（React）：**
```jsx
// Lucide 支持 fill 属性切换
import { Search, Bookmark } from 'lucide-react';

// 未选中
<Search stroke="#8C9096" strokeWidth={1.5} fill="none" />
// 选中
<Search stroke="#1A1C1E" strokeWidth={1.5} fill="#1A1C1E" />
```

- 侧边栏底色 `#FAFBFC`，右边框 `#E8EAED`
- Logo 区：纯文字「Signal」，衬线体，无图标
- 导航项：选中项左侧 3px 黑色竖线 + 文字 `#1A1C1E`，未选中 `#686C72`
- 导航项 hover：文字变深色
- 不折叠、不 hover 展开，固定宽度
