## 一、设计定位

### 产品类型：AI 行业资讯 / 新闻聚合 Dashboard

### 推荐风格：Editorial Grid / Magazine + Bento Box Grid（融合）

| 维度 | 选择 | 理由 |
|------|------|------|
| **主风格** | Editorial Grid / Magazine | 日报产品本质是「每日阅读」，适合杂志式排版，有节奏感、信息层级分明 |
| **辅风格** | Bento Box Grid | 统计卡片和概览区使用 Bento 布局，结构化展示数据 |
| **气质基调** | 专业 · 克制 · 深色质感 | 面向 AI 行业从业者，不是消费级娱乐产品 |
| **差异化记忆点** | 「可拖拽 AI 悬浮窗」+「自适应双栏阅读器」 | 这是你独有的功能，UI 应放大它 |

### 反模式（Avoid）

| ❌ 不要 | ✅ 要 |
|---------|------|
| 荧光蓝紫渐变色 | 低饱和冷色系 + 单色点缀 |
| 大面积纯色块 | 多层次透明度叠加 |
| 统一大小卡片堆叠 | 差异化信息密度的杂志式布局 |
| 内容贴边无呼吸感 | 清晰的分区 padding 体系 |
| 纯图标无文字标签 | 图标 + 文字联动标注 |

## 二、布局架构 — 「三栏非对称」解决左重右轻

### 当前问题
```
[侧边栏 64px] [内容区 100%] ← 右侧大量空白
```

### 新架构
```
┌──────┬────────────────────────┬──────────────┐
│      │                        │              │
│ 侧边  │  主内容区（~65%）        │  右侧面板     │
│ 栏    │                        │  (~30%)      │
│      │  Bento 统计卡片          │  热点关键词   │
│ 64px │  ────────────────────   │  ──────────  │
│ hover│  来源分组文章列表         │  趋势速览     │
│→220px│  差异化卡片密度          │  ──────────  │
│      │                        │  今日 Top 3   │
│      │                        │  ──────────  │
│      │                        │  AI 快捷提问   │
│      │                        │              │
├──────┴────────────────────────┴──────────────┤
│  底部: AI 浮动气泡（可拖拽）                   │
└──────────────────────────────────────────────┘
```

**关键变化：**
- 不再让内容区占满整个剩余宽度
- 右侧新增常驻面板（非抽屉），只在首页列表模式显示
- 进入阅读模式时，右侧面板自然过渡为「AI 精读 + 对话」
- 1440px 以上三栏布局，1024px 以下右侧面板折叠

### 响应式断点

| 断点 | 侧边栏 | 右侧面板 |
|------|--------|---------|
| ≥1440px | 64px hover→220px | 显示（380px） |
| ≥1024px | 64px hover→220px | 显示（340px） |
| ≥768px | 64px hover→220px | 折叠，用右上按钮展开 |
| <768px | 汉堡菜单 | 底部 sheet 弹窗 |

## 三、色彩体系（基于当前暗色 + 微调）

### 保留你现有的配色方案（经过之前优化已经很成熟）

```
背景层:  #0F1020  →  #16172D  →  #1A1B33  →  #22233D  →  #242542
         (最底层)    (统计卡)    (卡片)       (凸起)      (hover)
         
文字层:  #E2E6F9  →  #9197C2  →  #6E739C
         (标题)      (辅助)      (备注)

点缀色:  #6395FF（蓝）  #39C488（绿）  #F27070（红/高优）
         #D4A44A（黄/中优）            #6E739C（灰/低优）
         
边框:    #2E2F4F（统一）
```

### ✅ 当前配色优秀的地方
- 统一卡片底色 `#1A1B33`，不再深浅乱跳 ✅
- 放弃荧光色 ✅
- 三档文字色阶清晰 ✅
- 统计卡片底色统一深色、仅数字带色彩 ✅

### 🎯 需要改的一点
- 层级间距拉大：当前的 `#0F1020 → #16172D → #1A1B33` 明度差约 4%/3%，建议在卡片之间增加一个 `#13152A` 作为内容区底色，让卡片更「浮」出来

### 5.7 AI 浮动气泡

```
        ┌──────────────────────┐
        │  🤖 问我任何 AI 问题  │  ← 引导浮层（5s 后消失）
        └──────────────────────┘
                 ↑
            ┌──────────┐
            │  🤖       │  ← 可拖拽，#1A1B33 毛玻璃
            └──────────┘
```

- 定位：可拖拽，默认右下角
- 样式：`#1A1B33` + `backdrop-filter: blur(12px)` + 圆角 50%
- **拖拽反馈：** 拖拽开始 scale(1.12)，松手弹性回弹
- **自动隐藏：** 进入阅读模式自动隐藏，返回列表恢复
