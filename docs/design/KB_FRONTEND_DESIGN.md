## 2. 页面分区说明

### 2.1 顶部区域（Header）

| 元素 | 说明 |
|------|------|
| 标题 | 左侧「知识库」，使用 `Source Serif 4` 字体 |
| 统计 | 「共 N 个文档」，灰色小字 |
| 下载按钮 | 右上角，SVG 图标 + 「下载」文本 |
| 刷新按钮 | 右上角，SVG 图标 + 「刷新」文本 |

### 2.2 搜索栏

| 属性 | 值 |
|------|-----|
| 位置 | Header 下方 |
| 类型 | 文本框 + 搜索图标 |
| 行为 | 输入后 300ms 防抖自动搜索文件名 |
| 清空 | 右侧 X 按钮或 ESC 键 |

### 2.3 工具栏

| 区域 | 元素 | 说明 |
|------|------|------|
| 左侧 | 「+ 新增」按钮 | SVG 图标 + 文本，触发文件选择 |
| 左侧 | 「批量▾」下拉菜单 | SVG 图标 + 文本，含多个批量操作项 |
| 右侧 | 「每页显示: 20 条」 | 下拉选择 10/20/50 条 |

**「批量▾」下拉菜单选项**：

| 操作 | 说明 | 触发条件 |
|------|------|----------|
| 批量删除 | 删除选中的文档 | 至少选中 1 项 |
| 批量处理 | 处理选中的待处理文档 | 至少选中 1 项待处理文档 |
| 批量下载 | 下载选中的文档 | 至少选中 1 项 |
| 全选/取消全选 | 切换所有复选框状态 | 始终可用 |

### 2.4 筛选栏

| 筛选项 | 类型 | 选项 |
|--------|------|------|
| 分类（文件类型） | 下拉菜单 | 全部 / TXT / Markdown / PDF / DOCX |
| 状态（切片状态） | 下拉菜单 | 全部 / 待处理 / 处理中 / 已完成 / 失败 |
| 来源 | 下拉菜单 | 全部 / 网站 / 用户上传 |
| 查询按钮 | 按钮 | SVG 图标 + 「查询」文本 |
| 重置按钮 | 按钮 | SVG 图标 + 「重置」文本 |

**交互逻辑**：
- 文件名搜索 = 实时（独立触发）
- 筛选条件 = 点击「查询」才触发
- 多条件组合 = AND（且）关系
- 「重置」= 清空所有条件并重新加载

### 2.5 表格

| 列 | 宽度 | 说明 |
|----|------|------|
| ☑ | 40px | 复选框（批量选择） |
| # | 50px | 自增序号 `(page-1)*page_size + index` |
| 名称 | 25% | 文件名，**可点击打开预览抽屉** |
| 类型 | 8% | 文件类型缩写（PDF/MD/TXT/DOCX） |
| 来源 | 8% | 来源标签（网站/用户） |
| 命中次数 | 8% | 搜索命中次数，未处理显示 `-` |
| 状态 | 18% | 中文标签 + 颜色徽章 + 切片数（如 `已完成(40)`） |
| 操作 | 18% | 图标按钮 + [···更多] 下拉菜单 |

### 2.6 状态列显示格式

| 状态 | 显示格式 | 示例 |
|------|----------|------|
| pending | 待处理 | `⬜ 待处理` |
| processing | 处理中(当前/总数) | `⏳ 处理中(15/40)` |
| completed | 已完成(切片数) | `✅ 已完成(40)` |
| failed | 失败 | `❌ 失败` |

### 2.7 状态徽章配色

| 状态 | 标签 | 背景色 | 文字色 |
|------|------|--------|--------|
| pending | 待处理 | `#FFF3E0` | `#E65100` |
| processing | 处理中 | `#E3F2FD` | `#1565C0` |
| completed | 已完成 | `#E8F5E9` | `#2E7D32` |
| failed | 失败 | `#FFEBEE` | `#C62828` |

### 2.8 操作菜单

**可见按钮（直接展示，仅 SVG 图标）**：
- 图谱 — 网络图标（仅 completed 时可用）
- 预览 — 眼睛图标（始终可用）

**更多菜单 [···] 含**：

| 操作 | 说明 | 可用状态 |
|------|------|----------|
| 重试 | 重新处理文档 | 仅 failed |
| 修改标签 | 弹窗编辑标签 | always |
| 下载 | 下载原文件 | always |
| 删除 | 删除文档 | always |

**按钮状态规则**：

| 操作 | pending | processing | completed | failed |
|------|---------|------------|-----------|--------|
| 图谱(图标) | ❌ | ❌ | ✅ | ❌ |
| 预览(图标) | ✅ | ✅ | ✅ | ✅ |
| 重试(...) | ❌ | ❌ | ❌ | ✅ |
| 修改标签(...) | ✅ | ✅ | ✅ | ✅ |
| 下载(...) | ✅ | ✅ | ✅ | ✅ |
| 删除(...) | ✅ | ✅ | ✅ | ✅ |

### 2.9 分页

| 元素 | 说明 |
|------|------|
| 左侧 | 「第 X/Y 页，共 N 个文档」 |
| 右侧 | 页码按钮：< 1 2 3 4 5 > |
| 当前页 | 高亮显示 |
| 省略 | 超过 7 页时用 ... 省略 |

## 4. 交互细节

### 4.1 文件上传

```
点击 [+ 新增] → 弹出文件选择器（支持多选）
    或者
拖拽文件到表格区域 → 显示上传进度

支持格式: .txt .md .pdf .docx
大小限制: 10MB
```

### 4.2 预览文档（点击文件名或预览按钮触发）

```
点击「预览」或「文件名」→ 右侧滑出预览抽屉
  ┌─ 预览 ──────────────────┐
  │  × 关闭  文档名  [下载]  │
  ├──────────────────────────┤
  │                         │
  │   文件内容渲染           │
  │   - TXT/MD: 直接渲染    │
  │   - PDF: iframe 嵌入     │
  │   - DOCX: 转换为 HTML   │
  │                         │
  │                         │
  └──────────────────────────┘
```

### 4.3 知识图谱

```
点击「图谱」→ 右侧滑出图谱抽屉（现有组件复用）

图谱抽屉包含：
  - 标题: 文档名 + 实体/关系数量
  - Canvas 力导向图
  - 图例（底部）
  - 关闭按钮
```

### 4.4 修改标签

```
点击「修改标签」→ 弹窗编辑

  ┌─ 编辑标签 ──────────────┐
  │  LLM  ×  测试  ×        │
  │  [输入标签...] [+ 添加]  │
  │                         │
  │           [取消] [保存]  │
  └──────────────────────────┘
```

### 4.5 批量操作

```
1. 勾选需要操作的文档（复选框）
2. 点击顶部「批量▾」下拉菜单
3. 选择操作类型：

   ┌─────────────────────┐
   │ 批量删除            │
   │ 批量处理            │
   │ 批量下载            │
   │ ─────────────       │
   │ 全选 / 取消全选     │
   └─────────────────────┘

4. 确认对话框（删除操作需要二次确认）
5. 执行 → 显示进度 → 刷新列表
```

### 4.6 重试

```
点击「重试」→ 对 failed 状态的文档重新调用处理接口
  → 状态变更为 processing
  → 处理完成后更新为 completed 或 failed
```

## 5. 新增后端字段

### 5.1 `kb_documents` 表新增字段

```sql
ALTER TABLE kb_documents ADD COLUMN IF NOT EXISTS source VARCHAR(50) DEFAULT 'website';
-- source: 'website'（网站）| 'user'（用户上传）

ALTER TABLE kb_documents ADD COLUMN IF NOT EXISTS hit_count INTEGER DEFAULT 0;
-- hit_count: 搜索命中次数

ALTER TABLE kb_documents ADD COLUMN IF NOT EXISTS chunks_count INTEGER DEFAULT 0;
-- chunks_count: 切片总数（冗余字段，方便列表展示）
```

### 5.2 新增 API 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/kb/documents/{id}/preview` | GET | 获取文档内容（预览） |
| `/api/kb/documents/{id}/download` | GET | 下载原文件 |
| `/api/kb/documents/{id}/tags` | PUT | 修改文档标签 |
| `/api/kb/batch/delete` | POST | 批量删除文档 |
| `/api/kb/batch/process` | POST | 批量处理文档 |
| `/api/kb/batch/download` | POST | 批量下载（返回 zip） |
