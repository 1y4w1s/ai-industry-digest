# 知识图谱

## 位置

`api/routes/kb.py`

## 全流程

```
用户上传 PDF/DOCX/TXT/MD
    │
    ▼
文件校验 (扩展名 / 大小 10MB / 类型)
    │
    ▼
文本提取 + 切片 (chunking)
    │
    ▼
AI 实体识别 + 关系抽取 (DeepSeek)
    │
    ▼
存入图谱表 (entities + relations)
    │
    ▼
前端 Canvas 力导向图可视化
```

## AI 实体识别

```python
prompt = f"""从以下文本中提取实体和关系：
文本：{chunk_text}
输出 JSON 格式：
{{
    "entities": [
        {{"name": "OpenAI", "type": "company"}},
        {{"name": "GPT-4", "type": "product"}}
    ],
    "relations": [
        {{"source": "OpenAI", "target": "GPT-4", "relation": "开发"}}
    ]
}}"""

response = deepseek.chat(prompt)
entities, relations = parse_json(response)
```

实体类型支持：`company`, `person`, `tech`, `product`, `concept`

## 数据库存储

```sql
-- 实体表
CREATE TABLE kb_entities (
    id UUID PRIMARY KEY,
    document_id UUID REFERENCES kb_documents,
    name TEXT NOT NULL,
    entity_type TEXT NOT NULL
);

-- 关系表
CREATE TABLE kb_relations (
    id UUID PRIMARY KEY,
    document_id UUID REFERENCES kb_documents,
    source_entity_id UUID REFERENCES kb_entities,
    target_entity_id UUID REFERENCES kb_entities,
    relation_type TEXT NOT NULL
);
```

## 前端可视化

使用 Canvas 自实现力导向图，未使用 D3.js 等第三方库：

```javascript
function drawGraph(entities, relations) {
    // 节点初始化：随机位置
    nodes = entities.map(e => ({ ...e, x: rand(), y: rand() }));

    // 迭代力导向布局
    for (let i = 0; i < 100; i++) {
        // 斥力：同类型节点互相排斥
        // 引力：有关系节点互相吸引
        // 阻尼：速度衰减
        updateForces(nodes, relations);
    }

    // Canvas 渲染
    ctx.clearRect(0, 0, width, height);
    drawEdges(ctx, relations, nodes);   // 画边
    drawNodes(ctx, nodes);              // 画节点
}
```
